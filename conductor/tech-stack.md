# Technology Stack

This document outlines the core technologies used in the project.

## Core Frameworks & Languages
- **Language:** [TypeScript](https://www.typescriptlang.org/) (Strict Mode)
- **Frontend Framework:** [Angular v21+](https://angular.dev/) (Standalone Components, Signals)
- **State Management:** Angular Signals

## UI & Styling
- **Utility CSS:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Component Library:** [PrimeNG v21+](https://primeng.org/)
- **Design System:** PrimeUI Themes
- **Icons:** [Lucide Angular](https://lucide.dev/), [PrimeIcons](https://primeng.org/icons)

## Tooling & Infrastructure
- **Build System:** Angular CLI / Vite
- **Unit Testing:** [Vitest](https://vitest.dev/)
- **Package Manager:** [npm](https://www.npmjs.com/)
